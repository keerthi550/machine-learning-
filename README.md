# CNN Feature Map Visualisation Tutorial

This repository contains the code and tutorial for exploring how Convolutional Neural Networks (CNNs) learn visual features using the Fashion-MNIST dataset.

## Project Overview

The tutorial demonstrates:

- Building and training a CNN with TensorFlow/Keras
- Evaluating classification performance
- Visualising feature maps from convolutional layers
- Analysing correct and incorrect predictions
- Understanding how CNNs learn hierarchical image representations

## Dataset

Fashion-MNIST Dataset

Classes include:

- T-shirt/top
- Trouser
- Pullover
- Dress
- Coat
- Sandal
- Shirt
- Sneaker
- Bag
- Ankle boot

## Requirements
- Python 3.x
- TensorFlow
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
